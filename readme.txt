== Description ==
**گرویتی فرم فارسی**
بسته کامل فارسی ساز گرویتی فرم
با کمک افزونه گرویتی فرمز فارسی می توانید تمامی بخش های گرویتی فرم را فارسی کنید، فیلد کد ملی را اضافه کنید و همچنین لیست شهر ها و استان های ایران را به فرم‌های گرویتی فرم اضافه نمایید.
همچنین این بسته حاوی بومی ساز کامل گرویتی فرم می باشد

= Features =
* بسته کامل ایرانی ساز گرویتی فرم
* ترجمه گرویتی فرم و افزونه های جانبی
* ارسال پیامک در رویدادهای مختلف
* راستچین سازی مدیریت و فرم ها
* استان‌های ایران
* شهرهای ایران
* فیلد کد ملی هوشمند
* انتخابگر تاریخ و تاریخ شمسی
* پیش نیاز و هسته مرکزی درگاه‌های پرداخت
* واحد پولی ریال و تومان
* ابزارهای بکار رفته کاربردی
* امکانات فراوان دیگر

= Compatibility =
* هسته گرویتی فرم نسخه 2.9.1 به بالا

== Frequently asked questions ==

= راهنمای نصب =
1. ابتدا هسته gravity forms نسخه 2.9.1 به بالا را نصب نمایید
2. سپس بسته فارسی ساز را یا از مخزن یا به صورت دستی همانند سایر افزونه‌ها نصب نمایید
3. افزونه را فعال نموده و لذت ببرید

= آیا گرویتی فرم در ایران سایت پشتیبانی دارد =
بله، می‌توانید به سایت گرویتی فرم فارسی مراجعه نمایید [GravityForms.ir](https://gravityforms.ir)

= آیا سایت گرویتی فرم فارسی، هسته گرویتی فرم را نیز به فروش می رسانید =
خیر به هیچ وجه! برای خرید گرویتی فرم تنها از طریق سایت رسمی آن خرید نمایید GravityForms.com
